
# MyInfo: Improved Student Portal for Forth Valley College 🎓

This project is a modern reimagining of a College student portal, *MyInfo*. Built as part of a student-centred redesign initiative, the portal improves usability, accessibility, and overall functionality — ensuring students can more easily manage academic and personal tasks online.

🚀 **[Live Demo](https://myinfo.rhysjpaterson.com)**  
🧑‍💻 **Developed by:** Rhys John Paterson

---

## 📌 Overview

Forth Valley College is a leading educational institution in Scotland. Their current portal, *MyInfo*, is a key resource for students but has notable usability and functionality issues. This project aims to resolve those issues by building a clean, intuitive, and responsive web-based portal that meets both academic and administrative needs.

---

## 🎯 Purpose

The improved portal aims to:
- Deliver a more user-friendly and intuitive interface.
- Enhance access to academic resources and tools.
- Simplify common tasks like enrolment, attendance tracking, and timetable viewing.

---

## 🚦 Core Features

- **Online Enrolment**: Digital form submission for streamlined registration.
- **Attendance Tracking**: View attendance history and submit self-certification.
- **Timetabling**: Access class names, times, locations, and lecturer info.
- **Contact Information Management**: Update emergency contacts and set communication preferences.
- **Parking & ASN Requests**: Submit requests directly through the portal.
- **Password Reset Support**: Request password assistance easily.
- **Student Profile Updates**: View and request changes to personal data, including photos.
- **College Info Hub**: Find campus contact details quickly.
- **Quick Links**: Jump to Moodle, funding pages, FVSA, and more.
- **Secure Login**: Built-in user authentication for protected access.

---

## 🧩 Database Design (Entity Overview)

- **Students**: Holds personal info, contact preferences, emergency contacts, and login credentials.
- **Classes**: Contains class schedules, locations, and lecturer links.
- **Attendance**: Tracks student presence per class.
- **Lecturers**: Stores lecturer profiles and class associations.
